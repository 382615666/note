# robots.txt

* 必须在网站根目录
* User-agent 允许的搜索引擎
* Allow 允许的路径
* Disallow 禁止的路径
* [参考链接](https://blog.csdn.net/fanghua_vip/article/details/79535639)
* [参考链接](https://en.wikipedia.org/wiki/Robots_exclusion_standard)
```html
    <meta name="robots" content="index|noindex|follow|nofollow" />
    <a href="#" rel="nofollow"></a>
```
